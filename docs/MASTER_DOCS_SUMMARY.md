## Key Decisions Committed

### Rendering: Raycast 3D (COMMITTED)
- **Vision**: Raycasted 3D (Wolfenstein-style) with modern smoothing
- **Current**: 2D tile-based (Phaser 3) - interim foundation
- **Status**: Vision committed, implementation Stage 3+ (after performance validation)

### Brand Identity
- **Name**: Ebb & Bloom (confirmed)
- **Tagline**: **"Shape Worlds. Traits Echo. Legacy Endures."** (Hybrid: gameplay clarity + poetic resonance)
- **Brand Voice**: Hybrid - Poetic for world/atmosphere, technical for systems, playful for onboarding

### Technology Stack (Locked)
- **Platform**: Ionic/Vue/Capacitor (mobile-first)
- **ECS**: BitECS (core architecture)
- **AI**: YukaJS (behavior profiling)
- **State**: Zustand (UI state management)
- **Rendering**: Phaser 3 (interim) → Raycast 3D (target)

### Goals
- **Primary**: Commercial release (app stores)
- **Target Audience**: Mobile-first gamers, 25-45, meditative/exploration-focused
- **Yuka Adaptation**: Adapts/reacts/steers based on player playstyle (respect vs abuse environment)
