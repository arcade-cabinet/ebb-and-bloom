# Intro Animation Assets

Place 2 animation variants here:
- animation-v1.mp4 (or .webm)
- animation-v2.mp4

**Specs**:
- Resolution: 1080x1920 (portrait) or 1920x1080 (landscape)
- Duration: 5-10 seconds max
- Format: MP4 (H.264) preferred
- Size: < 5MB each

See `/docs/ASSET_PLACEMENT.md` for details.
